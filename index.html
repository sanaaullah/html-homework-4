Most commonly used tags and elements in HTML:

A) !DOCTYPE

!DOCTYPE declares to the web browser about the html version 5.0

B) html

The tag represents the root of an HTML elements. And it is the cpntainer of all other HTML elements and tags except !doctype.

C) head

The HTML head element is a container for all other head elements:title, style, meta, link, base,script.

D) body

The body tag defines the documents body. This element contains all the contents of an HTML document, such as heading, paragraph,image, hyperlink, table,lists, etc.

E) h1-h6

The h1 to h6 elements used to define HTML headings.h1 defines the most important heading and h6 defines the least important heading.

F) p

The p tags defines a paragraph. Browser automatically add a single blank line before and after each p element.

G) a

The a tag defines a hyperlink, which used to link from one page to another. The most important of the a element is the href attribute, whichh indicates the links destination

H) div

The div tag defines a division or a section in an HTML document.The div tag is easily styled by using the class or id attriute.

I) nav

The nav tag defines a set of navigation links.

Notice that NOT all linksof a document should be inside a nav element.The nav element is intended only for major block of navigation link.
